Modulos

Um modulo é um mecanismo que agrupa componentes, diretivas, pipes de forma a combinar com outros módulos para criar um aplicativo.

Cada modulo tem como objetivo fornecer uma funcionalidade ou recurso espeficifico (Modelo de arquitetura de software).

@NgModule

declarations: componentes, diretivas ou pipes do modulo.

imports: importars, recursos de outro modulo.

exports: exportar componentes, diretivas ou pipes para outros modulos.

providers: adicionar recursos externos no modulo.

bootstrap: modulo principal do projeto. Declarado apenas no app.module.ts.

Como criar um Modulo?

Com Angular CLI, execute o comando ng generate module nome-module ou ng g m nome-module.

Angular Material

Angular Material é um framework Angular que disponibiliza componentes prontos para usar em aplicativos profissionais. Usamos os componentes do Angular Material importando o module referente aquele componente. Angular Material
