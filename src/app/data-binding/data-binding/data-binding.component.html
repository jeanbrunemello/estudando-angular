O que é Data Binding?

A vinculação de dados mantém sua página atualizada automaticamente com base no estado do seu aplicativo. Você usa vinculação de dados para especificar coisas como a origem de uma imagem, o estado de um botão ou dados de um usuário específico.

Tipos de Data Binding

Interpolation
Interpolation é do tipo one way data binding, ou seja, o dado é apenas apresentado na tela para o usuário, sem alteração do valor internamente. Para fazer essa interpolação, use chaves duplas.
Property Binding
One way data binding, o Property Binding faz vinculação de valores de atributos do componente aos atributos de um elemento HTML.
Event Binding
O Event Binding funciona para ouvir eventos nativos do HTML ou eventos que podem ser criados pelo programador, executando algum método do componente através de um evento acionado.
Two Way Data Binding
O Two Way Data Binding é um tipo especial de binding, onde além do valor do atributo ser apresentado na tela do usuário, caso o valor seja editado em algum campo de input, o valor será automaticamente alterado internamente no componente.
