HTTP

HTTP (Hypertext Transfer Protocol, ou Protocolo de Transferência de Hipertexto) trata-se do protocolo de comunicação fundamental da web. Ele permite executar requisições e envio de documentos (HTML, CSS, JavaScript) pela internet.

HttpClient

Angular fornece HttpClient, classe de serviço que fornece recursos para requisição e envio de dados via HTTP, que permite a comunicação da aplicação com uma API (back-end). Para utilizar esse serviço é necessário importar o modulo HttpClientModule.

Exemplo (Filmes)

Obs: Antes de executar a aplicação Angular, é necessário executar o json-server. (npm install json-server / json-server --watch server/films.json)
