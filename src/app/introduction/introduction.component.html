O que é Angular?

Angular é um framework TypeScript baseado em componentes. Ele é uma ferramenta de desenvolvimento para criar aplicativos de página única eficientes e sofisticados.

O que é TypeScript?

O TypeScript é uma linguagem de software livre desenvolvida pela Microsoft. TypeScript é uma liguagem baseada em JavaScript com sistema de tipagem embutido. "TypeScript é JavaScript com tipagem".

Como funciona o TypeScript?

O código TypeScript não pode ser interpretado pelo Node.js (interpretador de JavaScript) nem pelo navegaor, pois ambos só entendem JavaScript. Antes de ser executado o codigo fonte escrito em TypeScript é convertido em um código JavaScript, esse processo é chamado de transpilação.

Tipos em TypeScript

  • string (texto)
  • number (numero)
  • boolean (logico)
  • any (qualquer tipo)
  • null (nulo)
  • undefined (indefinido)
  • Array (vetor)
  • Object (objeto)

Angular CLI

A CLI Angular é uma ferramenta de interface de linha de comando que você usa para inicializar, desenvolver, estruturar e manter aplicativos Angular diretamente de um shell de comando. A instalação é feita através do NPM com o comando npm install -g @angular/cli.

Comandos Angular CLI

ng new [nome projeto]
Criar um novo projeto Angular.
ng serve --open
Executar o projeto Angular e abrir automaticamente no navegador.
ng generate component [nome do componente] / ng g c [nome do componente]
Criar um novo componente dentro do componente app.
