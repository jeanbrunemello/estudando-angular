O que é um Componente?

Um componente é um bloco de construção que compõe um aplicativo. Em angular um componente é formado por quatro arquivos: nome.component.html, nome.component.css, nome.component.spec.ts e nome.component.ts. A classe que representa um componente em Angular é anotada com o decorator @Component.

Como criar um Componente?

Com Angular CLI, execute o comando ng generate component nome-componente ou ng g c nome-componente.
