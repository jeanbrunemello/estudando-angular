Template Driven Forms (Formulário Orientado a Modelo)

Em um formulário orientado por modelo os elementos de controle no formulário estão vinculados a propriedades de dados que possuem validação de entrada. A validação de entrada ajuda a manter a integridade e o estilo dos dados para melhorar a experiência do usuário. Os formulários controlados por modelo usam a vinculação de dados bidirecional (two way data binding) para atualizar o modelo de dados no componente à medida que as alterações são feitas no modelo ou no template. Para utilizar formulários baseado em modelo deve-se importar o modulo FormsModule.

NgForm

Cria uma instância de FormGroup e a vincula a um elemento form para rastrear o valor agregado do formulário e o status de validação. Assim que você importa FormsModule, esta diretiva se torna ativa por padrão em todas as tags form.

NgModel

Vincula alterações de valor no elemento de formulário anexado com alterações no modelo de dados, permitindo que você responda à entrada do usuário com validação de entrada e tratamento de erros.
