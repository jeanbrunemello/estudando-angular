Pipes

Funções simples para usar em expressões de template que aceita um valor de entrada e retorna um valor tranformado.

uppercase

Transforma todas as letras de uma string em maiúsculas.

lowercase

Transforma todas as letras de uma string em minúsculas.

number

Formata números espicificando por parâmetro a quantidade mínima de números inteiros e a quantidade mínima e maxima de numeros decimais.

date

Formata um objeto do tipo Date a partir do modelo de data passado pôr parâmetro.

json

Converte um objeto para um modelo de visualização Json.

Pipes Customizadas

É possível criar Pipes que realizam processos customizados. Para criar uma nova Pipe execute o comando ng generate pipe nome-pipe ou ng g p nome-pipe.

Musica

Nome: {{ music.name | uppercase }}

Autor: {{ music.author | lowercase }}

Numero de visualizações: {{ music.view | number:"1.1-2" }} bilhões

Data de lançamento: {{ music.lancamento | date:"dd/MM/yyyy" }}
