Servies

Um Service Angular é uma objeto que fornece fornece funções para manipulação de dados. Essas funções podem ser chamadas pelos componentes da aplicação. Os Services encapsulam os recursos da aplicação para que possam ser utilizados em multiplos modulos. Para criar um novo Service com Angular CLI execute o comando ng generate service [nome do serviço] / ng g s [nome do serviço].
